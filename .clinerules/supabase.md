# Supabase Guidelines

## Core Documentation

For any tasks involving Supabase, refer to the official documentation as your primary source.

- **pgvector Extension Overview**: https://supabase.com/docs/guides/database/extensions/pgvector
- **Detailed Tutorial (including query function)**: https://supabase.com/blog/openai-embeddings-postgres-vector
- **Supabase JavaScript Client**: https://supabase.com/docs/reference/javascript/introduction
- **Creating Database Functions**: https://supabase.com/docs/guides/database/functions

## Instructions

- Adhere strictly to the configuration options and API usage described in the above linked documentation.
- Do not use deprecated or non-existent configuration options from your training data. The URLs above contain the most current information.
- If you get 404 errors by visiting the above URL links, please stop and report. You'll be given an instruction on where to look at instead.
