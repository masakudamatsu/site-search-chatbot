# Vercel AI SDK Guidelines

## Core Documentation

For any tasks involving the Vercel AI SDK, refer to the official documentation as your primary source.

- **Core Concepts & Foundations**: https://ai-sdk.dev/docs/foundations
- **Getting Started with Next.js App Router**: https://ai-sdk.dev/docs/getting-started/nextjs-app-router
- **Backend: Generating Text & Streaming**: https://ai-sdk.dev/docs/ai-sdk-core/generating-text
- **Frontend: Building a Chatbot UI**: https://ai-sdk.dev/docs/ai-sdk-ui/chatbot
- **Full API Reference**: https://ai-sdk.dev/docs/reference
- **useChat hook**: https://ai-sdk.dev/docs/reference/ai-sdk-ui/use-chat#usechat
- **Migrate AI SDK 4.0 to 5.0 for useChat hook**: https://ai-sdk.dev/docs/migration-guides/migration-guide-5-0#usechat-changesmigration-guide-5-0#usechat-changes
- **UIMessage type**: https://ai-sdk.dev/docs/reference/ai-sdk-core/ui-message#uimessage
- **`streamText`**: https://ai-sdk.dev/docs/reference/ai-sdk-core/stream-text
- **Together.ai Provider**: https://ai-sdk.dev/providers/ai-sdk-providers/togetherai

## Instructions

- Adhere strictly to the API usage and patterns described in the above linked documentation.
- The AI SDK is under active development, so it is critical to follow the latest documentation to avoid using outdated or deprecated features.
- If you get 404 errors by visiting the above URL links, please stop and report. You'll be given an instruction on where to look at instead.
