# Replicate Guidelines

## Core Documentation

For any tasks involving Replicate, refer to the official documentation as your primary source.

- **How does Replicate work?**: https://replicate.com/docs/reference/how-does-replicate-work 
- **Run a model with Node.js**: https://replicate.com/docs/get-started/nodejs
- **Build a Next.js web app that uses Replicate to run models**: https://replicate.com/docs/guides/run/nextjs

## Instructions

- Adhere strictly to the configuration options described in the above linked documentation.
- Do not use deprecated or non-existent configuration options from your training data. The URLs above contain the most current information.
- If you get 404 errors by visiting the above URL links, please stop and report. You'll be given an instruction on where to look at instead.
