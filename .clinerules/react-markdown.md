# React Markdown Guidelines

## Core Documentation

For any tasks involving the `react-markdown` library, refer to the official documentation as your primary source.

- **Documentation**: https://github.com/remarkjs/react-markdown
- **Demo**: https://remarkjs.github.io/react-markdown/
- **Remark GFM plugin**: https://github.com/remarkjs/remark-gfm

## Instructions

- Adhere strictly to the configuration options described in the above linked documentation.
- Do not use deprecated or non-existent configuration options from your training data. The URLs above contain the most current information.
- If you get 404 errors by visiting the above URL links, please stop and report. You'll be given an instruction on where to look at instead.
